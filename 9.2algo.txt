Algorithm: Non-Preemptive Priority Scheduling

Step 1: Start

Step 2: Read number of processes n.

Step 3: For each process i = 1 to n, read:

Arrival Time AT[i]

Burst Time BT[i]

Priority PR[i]

Set:

completed = 0

time = 0

Step 4: Repeat until completed = n:

    4.1 Set idx = -1
    4.2 Set max_priority = -∞

    4.3 For each process i = 1 to n
        If:

AT[i] ≤ time

Process is not completed

        Then:

If PR[i] > max_priority

max_priority = PR[i]

idx = i

    4.4 If idx = -1

time = time + 1

Go to Step 4

    4.5 Else:

time = time + BT[idx]

CT[idx] = time

TAT[idx] = CT[idx] - AT[idx]

WT[idx] = TAT[idx] - BT[idx]

Mark process as completed

completed = completed + 1

Step 5: Compute:

Average Turnaround Time = ΣTAT / n

Average Waiting Time = ΣWT / n

Step 6: Stop
