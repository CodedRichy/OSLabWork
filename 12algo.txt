ALGORITHM: Deadlock-Free Dining Philosophers using Semaphores

Step 1: Declare Constants and Variables
1. Define constant N = 5.
2. Declare semaphore room.
3. Declare semaphore chopstick[0 … N-1].
4. Declare semaphore mutex.
5. Declare thread array tid[0 … N-1].

Step 2: Initialize Semaphores
6. Initialize room ← N - 1.
7. For i = 0 to N - 1
      initialize chopstick[i] ← 1.
8. Initialize mutex ← 1.

Step 3: Create Philosopher Threads
9. For i = 0 to N - 1
      create thread tid[i] executing Philosopher(i).

Step 4: Philosopher Execution
10. Repeat forever:
11.     Philosopher thinks.
12.     wait(room).
13.     wait(chopstick[id]).
14.     wait(chopstick[(id + 1) mod N]).
15.     wait(mutex).
16.     Philosopher eats.
17.     signal(mutex).
18.     signal(chopstick[id]).
19.     signal(chopstick[(id + 1) mod N]).
20.     signal(room).

Step 5: Wait for Completion
21. Main thread waits for all philosopher threads.
